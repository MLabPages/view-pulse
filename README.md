# ViewPulse

外カメで「見ていた場面」を撮影しながら、内カメで表情とおおまかな視線を端末内解析するスマートフォン向けWebアプリです。

## 主な機能

- 通常の動画カメラに近い全画面撮影UI
- 前後カメラの同時利用（対応端末のみ）
- MediaPipe Face Landmarkerによる表情・顔向き・視線の端末内解析
- 5点の視線キャリブレーション
- 外カメ動画と視線・表情データの時刻同期
- 撮影後の視線ヒートマップと反応タイムライン
- 任意同意時のみ、表情映像を保存
- 表情を大きく、見ていた場面を小窓にしたリアクション映像
- 外カメ動画、分析JSON、リアクション映像の端末保存

## プライバシー

- 映像と分析はブラウザ内で完結し、サーバーへ送信しません。
- 内カメ映像は、利用者が「表情映像も端末内に保存する」を選んだ場合だけ記録します。
- 選ばない場合、内カメ映像はリアルタイム解析後に破棄され、数値だけが残ります。
- ページを再読み込みすると、保存していない映像とデータは消えます。
- MediaPipeのモデルファイルを読み込むため、初回のみGoogle／jsDelivrのCDNへ接続します。映像は送信しません。

## 起動

カメラ利用にはHTTPSまたはlocalhostが必要です。

```powershell
node local-server.mjs
```

PCでは `http://127.0.0.1:8765`、スマートフォンではHTTPSで公開したURLを開きます。

## 検証

```powershell
node --check app.js
node --check local-server.mjs
node verify.mjs
```

## 端末上の制約

Webブラウザで前後カメラを同時に利用できるかは端末・OS・ブラウザに依存します。非対応の場合は外カメ動画だけを撮影する `SCENE ONLY` モードへ自動的に切り替わります。

ヒートマップは専用アイトラッカーではなく内カメからの近似です。利用者がスマートフォン画面をファインダーとして見ている条件で、画面上の大まかな注目領域を確認する用途に向きます。研究利用では自己報告などとの併用を推奨します。
